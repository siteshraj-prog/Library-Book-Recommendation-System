Library Book Recommendation System – Project Statement

#Problem Statement
Readers often face difficulty choosing books that match their interests due to the vast number of available options. Libraries and digital reading platforms need simple tools to help users quickly find relevant books without manually searching through long lists. The goal is to build a system that recommends books based on a user’s preferred genre or reading preference.

#Scope of the Project
1.The system provides book recommendations based on predefined genres.
2.User selects a genre and receives a list of suggested books.
3.The project focuses on console-based interaction for simplicity.
4.Uses Python lists and dictionaries for data storage.
5.The recommendation logic is rule-based (no ML included).
6.Easily extendable to include more genres, books, or additional logic.

#Target Users
1.Students looking for books related to their interests.
2.Library visitors seeking quick suggestions.
3.Beginner programmers learning recommendation logic.
4.Academic projects or mini-projects for programming courses.
5.Anyone who wants personalized book suggestions using a simple tool.

# High-Level Features
1.Displays available genres to the user.
2.Accepts user input for selecting preferred genre(s).
3.Provides a curated list of recommended books for the chosen genre.
4.Handles invalid inputs gracefully.
5.Simple modular code structure for easy updates.

#Can be expanded to support:
1.Multiple recommendation criteria
2.User profiles
3.File/database-based book storage
